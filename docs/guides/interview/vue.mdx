## Vue2 的 $set
在 Vue 2 中，响应式系统有一定的限制，当你向一个已有响应式对象中 添加新的属性 时，Vue 无法检测到这个新属性的变化。

为了解决这个问题，Vue 提供了一个辅助方法：

```js
Vue.set(target, propertyName/index, value)
```

组件中使用：

```js
this.$set(this.someObject, 'newProperty', 'newValue')
```

使用场景说明：

```js
this.someObject = {
  ...this.someObject,
  newProperty: 'newValue'
}
## Vue2 Vue3区别
Suspense Fragment 组件
Vue的Suspense组件是Vue 3引入的一个特性，主要用于处理异步组件的加载状态。它的主要功能和用途包括：

1. 处理异步组件 - Suspense可以等待异步组件或异步setup()函数完成加载后再显示内容。
2. 提供加载状态 - 它有两个插槽：默认插槽用于展示异步完成后的内容，fallback插槽用于展示加载中的状态。
3. 简化错误处理 - 结合errorCaptured和onErrorCaptured可以更好地处理异步加载过程中的错误。
```vue
<template>
  <Suspense>
    <!-- 默认插槽：异步组件 -->
    <template #default>
      <AsyncComponent />
    </template>
    
    <!-- fallback插槽：加载状态 -->
    <template #fallback>
      <div>正在加载中...</div>
    </template>
  </Suspense>
</template>
```
