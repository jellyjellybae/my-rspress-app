`c`：删除并进入 insert 模式
`ch/dh` 删除当前光标前一个字符

`cj/dj` 删除当前光标所在行和下一行(一下子删两行)

- `d g _`：删除到行尾

- `S`：删除当前光标所在行的除缩进外的所有内容并进入 insert modec
